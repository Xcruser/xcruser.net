# Xcruser.net

Diese Dokumentation beschreibt den Aufbau und die Funktionsweise meiner persönlichen Website.

## Technologie-Stack

- **Next.js**: Framework für serverseitiges React
- **TypeScript**: Typsicheres JavaScript
- **Tailwind CSS**: Utility-First CSS Framework
- **Markdown**: Für die Dokumentation

## Features

### Status-Seite
Die Status-Seite zeigt Informationen über die Verfügbarkeit und Performance der Website:

- Echtzeit-Überwachung der Website-Verfügbarkeit
- Performance-Metriken und Ladezeiten
- Uptime-Tracking und Systemzustand

### Dokumentation
Die Dokumentationsseite bietet:

- Hierarchische Navigation
- Volltextsuche
- Markdown-Support
- Dark/Light Mode

## Entwicklung

### Lokale Entwicklung

1. Repository klonen:
\`\`\`bash
git clone https://github.com/yourusername/xcruser.net.git
\`\`\`

2. Dependencies installieren:
\`\`\`bash
npm install
\`\`\`

3. Development Server starten:
\`\`\`bash
npm run dev
\`\`\`

### Deployment

Die Website wird automatisch via GitHub Actions deployed.
